# Examples

Sample problems for Frontier. Each is a loadable `problem.json` (definition) + `scores.json` (options + scores), with a paste-able prompt — one per target problem type.

- **[Portfolio optimization](investment_portfolio/)** — *investment portfolio construction.* Return vs volatility (covariance) vs yield across 30 ETFs, with scenarios.
- **[Marketing channel budget](channel_budget/)** — *budget / channel allocation.* Split spend across channels to balance conversions across campaign goals.
- **[Supplier selection](supplier_selection/)** — *vendor selection.* Split orders across suppliers, balancing cost vs reliability.
- **[cuOpt portfolio](cuopt_portfolio/)** — the portfolio problem via the opt-in GPU cuOpt QP backend (notebook).

See the [main README](../README.md) for setup and [architecture.md](../architecture.md) for technical reference.
